## 项目概述
MoneyPrinterTurbo 是一个基于 AI 大模型的一键式高清短视频生成工具。用户只需输入一个主题或文案，系统便会自动完成脚本撰写、语音合成、素材匹配与视频合成，最终输出带有字幕和背景音乐的成品视频。它解决了内容创作者在视频制作中重复劳动多、技能门槛高、耗时长的痛点，让短视频生产变得像打字一样简单。

## 核心功能
- **AI 脚本生成**：输入一个主题或关键词，利用大模型自动撰写视频旁白文案，支持多语言和风格定制。
- **智能配音**：调用 TTS 引擎将脚本转为自然语音，可调节语速、音色、情感，并自动生成配音时间轴。
- **素材聚合**：根据文案语义自动从免费图库（Pexels/Pixabay）搜索并下载高清图片或视频片段，作为画面填充。
- **自动字幕**：结合语音时间戳与文案，自动生成逐字或逐句字幕，支持多种字幕样式与位置。
- **视频合成**：将配音、素材、字幕、背景音乐进行时间轴对齐，渲染为完整的高清短视频（常见分辨率如 1080p）。
- **一键操作**：提供 Web 图形界面，用户无需编写代码，点击即可完成从创意到出片的全部流程。
- **配置扩展**：支持更换不同的 LLM、TTS 引擎和素材源，方便高级用户定制与集成。

## 架构设计
项目采用**模块化管道架构**，将视频生成拆分为多个独立阶段，通过配置串联：

1. **Web UI 层** (Gradio/Streamlit)：接收用户输入的主题、风格、语音选项等。
2. **任务调度器**：根据配置调用各核心模块，管理数据流转与异常处理。
3. **文本生成模块**：封装 LLM 接口（如 OpenAI），根据 prompt 模板生成视频脚本。
4. **语音合成模块**：将脚本分段发送至 TTS 服务，返回音频文件及单词级时间戳。
5. **素材搜索模块**：从脚本中提取关键词，并行调用图库 API 获取免版权素材，并下载到本地。
6. **字幕生成模块**：基于时间戳与文案，生成标准字幕文件（.srt）并应用样式。
7. **视频合成模块**：利用 MoviePy 编排画面（图片/视频序列）、音频、字幕和背景音乐，调用 FFmpeg 渲染输出。
8. **配置与资源管理**：通过 YAML 或 .env 管理 API 密钥、服务端点、模板路径等。

数据流大致为：用户输入 → Web 界面 → 生成脚本 → 并行[合成语音 | 搜索素材] → 组装时间线 → 合成视频 → 输出文件。**关键设计模式**：管道模式、策略模式（可替换的 LLM/TTS 提供者）、以及异步 IO 提升素材获取速度。
